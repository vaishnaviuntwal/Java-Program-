Description
This repository contains a collection of Java programs covering fundamental to advanced concepts of programming. It is designed to strengthen problem-solving skills and build a strong foundation in Java.

Features
Basic Java programs (loops, conditions, patterns)
Data Structures implementation (Stack, Queue, Arrays)
Algorithm-based problems (sorting, searching)
Mathematical programs (LCM, GCD, palindrome, etc.)
String manipulation programs
Practice problems for coding interviews

 Technologies Used
Java
OOP (Object-Oriented Programming)
Data Structures & Algorithms
